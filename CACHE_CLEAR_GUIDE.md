# 清除瀏覽器快取指南

## 問題：修改沒有在平板上生效

這通常是瀏覽器快取造成的。請按照以下步驟清除快取：

### iPad / Safari

**方法 1：強制重新整理（最快）**
1. 在網頁上，長按 🔄 重新整理按鈕
2. 選擇「重新載入並清除快取」
3. 或者：按住 Shift 鍵 + 點擊重新整理

**方法 2：清除所有快取**
1. 設定 → Safari
2. 捲動到最下方
3. 點擊「清除瀏覽記錄和網站資料」
4. 確認清除
5. 重新開啟網站

### Chrome / Edge

1. 按 **Ctrl + Shift + Delete**（Windows）
2. 或 **Command + Shift + Delete**（Mac）
3. 選擇「快取的圖片和檔案」
4. 點擊「清除資料」
5. 重新整理頁面（F5 或 Ctrl+R）

### 強制重新整理（所有瀏覽器）

- **Windows**: `Ctrl + Shift + R` 或 `Ctrl + F5`
- **Mac**: `Command + Shift + R`
- **iPad**: 長按重新整理按鈕

## 驗證方法

清除快取後，請檢查：

1. **標題大小**：
   - 舊版：1.5rem（較小）
   - 新版：2.2rem（明顯變大）

2. **按鈕大小**：
   - 舊版：0.9-1.1rem
   - 新版：1.3rem（更大更易點擊）

3. **查看 Console 日誌**：
   - 按 F12 或長按頁面選「檢查元素」
   - 切換到 Console
   - 應該看到：🔥 設定 Firestore 即時監聽...
